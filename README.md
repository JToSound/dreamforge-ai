# DreamForge AI

> "The first open-source AI system that thinks while it sleeps."

This repository hosts DreamForge AI, a hierarchical multi-agent framework for simulating human-like dreaming, including sleep architecture, neurochemistry, memory consolidation, dream content generation, metacognition, and phenomenological reporting.

## Status

- Core biophysical models (sleep cycle, neurochemistry, memory graph) implemented.
- Agent layer wiring in progress.
- Initial visualization modules and launch materials scaffolded.
