# DreamForge Editions & Pricing Hypothesis

| Capability | Community (OSS) | Pro | Enterprise |
|---|---|---|---|
| Core simulation API + dashboard | ✅ | ✅ | ✅ |
| Narrative quality scoring/report endpoints | ✅ | ✅ | ✅ |

## Pricing Hypothesis (for validation)
- **Pro**: seat-based monthly plan for creators/research teams.
- **Enterprise**: annual contract with security/compliance support.
- **Community**: free and open source, optimized for adoption and trust.

## Validation Signals
- Trial-to-active usage conversion
- Compare/report feature adoption
- Team collaboration activation
- Enterprise security feature requests per quarter
